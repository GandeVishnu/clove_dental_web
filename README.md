# Clove Dental Landing Page 

This project is a static implementation of a Clove Dental style landing page

## Structure

```text
my-landing-page/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── script.js
└── README.md
```

## Sections Implemented

1. Hero + top appointment form with stats bar
2. Types of Root Canal Treatments (two cards)
3. Real Smiles, Real Transformations (smile gallery)
4. What Our Patients Say (testimonial row)
5. Google Reviews + Safety, Precision, and Innovation (review cards + feature cards)
6. Frequently Asked Questions + footer

All sections are responsive across desktop, tablet, and mobile using CSS Grid/Flexbox and media queries.
